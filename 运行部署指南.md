# 纺织品缺陷检测系统 · 运行部署指南

> 本文档说明如何在本地环境中从零开始配置并运行该系统。

---

## 📋 环境要求

| 项目 | 要求 |
|------|------|
| 操作系统 | macOS / Windows / Linux |
| Python 版本 | **3.10**（推荐，项目基于此版本开发） |
| 包管理工具 | Anaconda / Miniconda |
| 硬件 | CPU 即可运行；有 NVIDIA GPU 可加速推理 |

---

## 第一步：配置 conda 镜像源（国内必做）

> 如果你的 conda 下载很慢或报错，需要先切换为国内镜像。

编辑 `~/.condarc` 文件，替换为以下内容：

```yaml
channels:
  - defaults
show_channel_urls: true
default_channels:
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/r
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/msys2
custom_channels:
  conda-forge: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  pytorch: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
```

> ⚠️ 注意：`~/.condarc` 中不要填写 `pypi.douban.com` 等 **pip 镜像地址**，那是给 pip 用的，conda 无法识别。

---

## 第二步：创建 Python 3.10 虚拟环境

```bash
# 创建名为 textile 的环境
conda create -n textile python=3.10 -y

# 激活环境
conda activate textile
```

激活成功后，命令行前缀会变为 `(textile)`。

---

## 第三步：安装项目依赖

```bash
# 进入项目根目录
cd /path/to/Textile_Defect_Detection

# 安装 requirements.txt 中的依赖
pip install -r requirements.txt

# 安装 ultralytics（YOLO 核心库）
pip install ultralytics
```

> 💡 **pip 加速（可选）**：如果安装慢，可以添加国内镜像：
> ```bash
> pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
> pip install ultralytics -i https://pypi.tuna.tsinghua.edu.cn/simple
> ```

---

## 第四步：初始化数据库（新环境必做）

> 如果项目目录下已存在 `db.sqlite3` 文件（含有测试账号），可跳过此步。

```bash
# 执行数据库迁移
python manage.py migrate
```

若需要创建管理员账号，在 Django shell 中执行：

```bash
python manage.py shell < newsite/createadmin.py
```

这将创建账号：用户名 `admin`，密码 `admin`。

---

## 第五步：启动 Web 服务器

```bash
python manage.py runserver
```

看到如下输出即表示启动成功：

```
Django version 5.x, using settings 'newsite.settings'
Starting development server at http://127.0.0.1:8000/
Quit the server with CONTROL-C.
```

## 🔍 功能操作流程

### 1. 普通用户 · 图片检测

```
登录（普通用户账号）
  ↓
进入主界面
  ↓
点击"上传图片"，选择纺织品图片
  ↓
调整置信度阈值（默认 0.5）
  ↓
点击"开始检测"
  ↓
查看标注结果图（显示缺陷位置、类型、置信度）
  ↓
左侧侧边栏可查看历史检测记录
```

### 2. 普通用户 · 视频检测

```
登录 → 主界面
  ↓
切换到"视频检测"Tab
  ↓
上传视频文件（支持 .mp4 等格式）
  ↓
系统逐帧检测，生成带标注的结果视频
  ↓
下载或预览检测结果
```

### 3. 管理员 · 后台管理

```
登录（admin 账号）
  ↓
自动跳转至"用户管理"页面
  ↓
可进行以下操作：
  ├── 用户管理：新增/编辑/删除用户，分配角色
  ├── 检测记录：查看所有用户的历史检测结果
  └── 知识库管理：发布、编辑纺织品缺陷相关知识文章
```

## 🔧 常见问题排查

### Q1：`conda create` 报 502 / 连接失败
**原因**：conda 镜像源配置错误或网络问题。  
**解决**：按本文「第一步」重新配置 `~/.condarc`。

### Q2：`python manage.py runserver` 报 `No module named 'django'`
**原因**：当前终端没有激活 textile 环境。  
**解决**：
```bash
conda activate textile
```

### Q3：检测时报 `No module named 'ultralytics'`
**解决**：
```bash
pip install ultralytics -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q4：结果图片中中文标签显示乱码/方块
**原因**：字体文件路径问题。  
**解决**：确保项目根目录下存在 `simhei.ttf` 文件（已包含在项目中）。

### Q5：上传图片后页面无响应
**原因**：首次检测需要加载 YOLO 模型，约需 5-15 秒，之后会缓存，响应变快。  
**解决**：耐心等待即可。

## 🚂 模型重新训练（可选）

如需用自己的数据重新训练模型：

```bash
# 确保在 textile 环境下
conda activate textile

# 执行训练脚本（训练参数可在脚本内修改）
python run_train.py
```

训练结果会保存在 `runs/` 目录下，将最优模型 `runs/.../weights/best.pt` 复制到 `models/best.pt` 即可替换使用。
